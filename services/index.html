Services

Krellian creates web apps with all the power of mobile and desktop apps. Services include user & task analysis, user-centered design, agile development and usability testing.

User & Task Analysis

Find out what users really need, not by asking them what they want, but by observing them carrying out tasks and analysing what they do and what they're trying to accomplish.

User-centered Web App Design

From an understanding of user needs, take a creative and methodical approach to designing a web app to meet those needs and provide an exceptional user experience.

Agile Web Development

A disciplined and lightweight iterative approach to software development using cutting edge web technologies.

Usability Testing

Test systems with real users at every stage of development to feed into the iterative process and make sure you're building what the user really needs.
