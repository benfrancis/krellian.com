Bespoke Web Appliances

Consulting Services

A web appliance is a connected device dedicated to running a single web application. That could be an interactive kiosk or digital sign with custom input/output devices, a touch screen control pad dedicated to a specific task or an on-premises application server serving a specific function for your business.

Krellian provides consultancy services to help you design, prototype and build your own web appliances, benefiting from a decade of experience in web applications, web-based operating systems and connected devices.
